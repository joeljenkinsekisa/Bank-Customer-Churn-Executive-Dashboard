# Bank Customer Churn Executive Dashboard

## Project Overview
This project analyzes customer churn and retention using SQL and Power BI.

SQL was used for:
- Data cleaning
- Duplicate removal
- Data transformation
- Customer segmentation
- Churn analysis

Power BI was used to build:
- Executive dashboard
- Churn dashboard
- Retention dashboard

## Tools Used
- SQL
- Power BI
- Excel

## Key Insights
- Germany records the highest churn risk.
- Customers aged 50-59 show the highest churn levels.
- Inactive customers churn significantly more.
- Customers with 3 and 4 products show critical churn levels.
